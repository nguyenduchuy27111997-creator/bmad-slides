B · SDD Là Gì

Spec-Driven Development (SDD)

Viết spec bằng AI, code bằng AI, nhưng CON NGƯỜI quyết định

SDD = phương pháp phát triển mà specification là SOURCE OF TRUTH duy nhất — tạo bởi AI, review & approve bởi human trước khi implement.

📄
SPEC FIRST
Không code khi chưa có spec approve. Spec là hợp đồng giữa bạn và AI.
🔗
CONTEXT IN FILES
Mọi context nằm trong file, không trong chat. Version-controlled, persistent, shareable.
👤
HUMAN-IN-THE-LOOP
AI đề xuất, human quyết định. Review gate tại mỗi bước quan trọng.
